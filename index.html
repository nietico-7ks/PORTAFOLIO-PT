Sintaxis de PHP

PHP (Hypertext Preprocessor) es un lenguaje de scripting del lado del servidor. El código PHP se incrusta en HTML usando las etiquetas <?php ... ?>. Todo script comienza con la etiqueta de apertura y cada instrucción termina con ; (punto y coma).
sintaxis.php
// Etiqueta de apertura obligatoria <?php // Toda instrucción termina en punto y coma echo "Hola, mundo!"; // PHP también puede mezclarse con HTML echo "<h1>Bienvenido</h1>"; // PHP distingue mayúsculas en variables, no en funciones $nombre = "Ana"; Echo($nombre); // válido, Echo == echo ?>
01

Comentarios

Los comentarios son texto ignorado por el intérprete PHP. Sirven para documentar el código, explicar la lógica o deshabilitar temporalmente instrucciones. PHP soporta tres estilos de comentarios.
comentarios.php
<?php // 1. Comentario de una línea (estilo C++) $x = 10; // esto es x # 2. Comentario de una línea (estilo Shell) $y = 20; # esto es y /* 3. Comentario de múltiples líneas muy útil para documentar */ $z = $x + $y; echo $z; // imprime: 30 ?>
// Una línea → Comenta hasta el final de la línea.

# Una línea → Equivalente al anterior, heredado de Perl/Shell.

/* Bloque */ → Abarca múltiples líneas, ideal para descripciones largas o bloques de código deshabilitados.
02

Variables

En PHP las variables se declaran con el prefijo $ seguido del nombre. Son tipado dinámico: no necesitas declarar el tipo, PHP lo infiere automáticamente. Los nombres son sensibles a mayúsculas ($Nombre ≠ $nombre).
variables.php
<?php // Declaración y asignación $nombre = "Carlos"; $edad = 25; $altura = 1.75; $activo = true; // Asignación por referencia (alias) $a = 100; $b = &$a; // $b apunta al mismo valor que $a $a = 200; echo $b; // imprime: 200 // Variables variables (nombre dinámico) $campo = "ciudad"; $$campo = "Bogotá"; echo $ciudad; // imprime: Bogotá // Constantes (sin $) define("PI", 3.14159); echo PI; // imprime: 3.14159 ?>
$variable $$variable_variable &$referencia define("CONSTANTE") const CONSTANTE

Tipos de Datos

PHP maneja 8 tipos primitivos. Los tipos escalares son los más usados en el día a día; los compuestos permiten estructuras más complejas; y los especiales representan casos únicos como la ausencia de valor o referencias a recursos externos.
TipoEjemploDescripción
int$x = 42;Número entero (positivo o negativo)
float$p = 3.14;Número decimal de punto flotante
string$s = "hola";Cadena de texto
bool$ok = true;Valor verdadero o falso
array$a = [1,2,3];Colección de valores
object$obj = new Clase();Instancia de una clase
null$v = null;Ausencia de valor
resourcefopen(...)Referencia a recurso externo
tipos.php
<?php $entero = 42; $decimal = 3.14; $texto = "Hola PHP"; $booleano= true; $nulo = null; // Verificar el tipo var_dump($entero); // int(42) var_dump($decimal); // float(3.14) gettype($texto); // "string" // Conversión de tipos (casting) $num = (int) "42abc"; // → 42 $str = (string) 3.9; // → "3.9" $flt = (float) "7.5x"; // → 7.5 ?>
04

Operadores

Los operadores permiten realizar cálculos, comparaciones y operaciones lógicas. PHP tiene un conjunto rico de operadores, incluyendo algunos únicos como el operador nave espacial (<=>) y el de fusión null (??).
operadores.php
<?php // ── ARITMÉTICOS ────────────────────────── echo 10 + 3; // 13 → suma echo 10 - 3; // 7 → resta echo 10 * 3; // 30 → multiplicación echo 10 / 3; // 3.33 → división echo 10 % 3; // 1 → módulo (resto) echo 2 ** 8; // 256 → potencia // ── ASIGNACIÓN ─────────────────────────── $x = 10; $x += 5; // $x = 15 $x -= 3; // $x = 12 $x *= 2; // $x = 24 $x /= 4; // $x = 6 $x .= " pts"; // concatena // ── COMPARACIÓN ────────────────────────── var_dump(5 == "5"); // true (igual valor) var_dump(5 === "5"); // false (igual valor Y tipo) var_dump(5 != 4); // true var_dump(3 <=> 5); // -1 (nave espacial) // ── LÓGICOS ────────────────────────────── var_dump(true && false); // false → AND var_dump(true || false); // true → OR var_dump(!true); // false → NOT // ── ESPECIALES ─────────────────────────── $val = null ?? "default"; // "default" (fusión null) $msg = 1 > 0 ? "Sí" : "No"; // ternario ?>
05

Estructuras de Control

Permiten tomar decisiones en el flujo del programa. El bloque if/elseif/else es la base; switch evalúa múltiples casos; y match (PHP 8) es más estricto y devuelve un valor directamente.
estructuras.php
<?php $nota = 85; // ── IF / ELSEIF / ELSE ─────────────────── if ($nota >= 90) { echo "Sobresaliente"; } elseif ($nota >= 70) { echo "Aprobado"; // ← se ejecuta esto } else { echo "Reprobado"; } // ── SWITCH ─────────────────────────────── $dia = "lunes"; switch ($dia) { case "lunes": case "martes": echo "Inicio de semana"; break; case "viernes": echo "¡Casi fin de semana!"; break; default: echo "Día normal"; } // ── MATCH (PHP 8+) ─────────────────────── $estado = 2; $texto = match($estado) { 1 => "Pendiente", 2, 3 => "En proceso", // ← coincide default => "Desconocido", }; echo $texto; // En proceso ?>
06

Bucles

Los bucles permiten repetir un bloque de código. PHP ofrece cuatro tipos: for (contador conocido), while (condición al inicio), do...while (condición al final, siempre ejecuta al menos una vez) y foreach (ideal para arrays).
bucles.php
<?php // ── FOR ────────────────────────────────── for ($i = 0; $i < 5; $i++) { echo $i . " "; // 0 1 2 3 4 } // ── WHILE ──────────────────────────────── $contador = 1; while ($contador <= 3) { echo "Vuelta " . $contador; $contador++; } // ── DO...WHILE ─────────────────────────── $x = 10; do { echo $x; // se ejecuta aunque $x > 5 $x++; } while ($x < 5); // ── FOREACH ────────────────────────────── $frutas = ["mango", "kiwi", "fresa"]; foreach ($frutas as $fruta) { echo $fruta . "\n"; } // ── CONTROL DE FLUJO ───────────────────── for ($i = 0; $i < 10; $i++) { if ($i == 3) continue; // salta la iteración 3 if ($i == 7) break; // detiene el bucle en 7 echo $i . " "; } // imprime: 0 1 2 4 5 6 ?>
07

Arrays

En PHP, los arrays son estructuras de datos muy versátiles: pueden ser indexados (claves numéricas automáticas), asociativos (claves personalizadas tipo clave-valor) o multidimensionales (arrays dentro de arrays). La función array() y la sintaxis [] son equivalentes.
arrays.php
<?php // ── ARRAY INDEXADO ─────────────────────── $colores = ["rojo", "verde", "azul"]; echo $colores[0]; // rojo $colores[] = "amarillo"; // agregar al final // ── ARRAY ASOCIATIVO ───────────────────── $persona = [ "nombre" => "Luis", "edad" => 28, "ciudad" => "Medellín", ]; echo $persona["nombre"]; // Luis // ── ARRAY MULTIDIMENSIONAL ─────────────── $matriz = [ [1, 2, 3], [4, 5, 6], ]; echo $matriz[1][2]; // 6 // ── FUNCIONES ÚTILES ───────────────────── count($colores); // cantidad de elementos sort($colores); // ordenar (modifica el original) array_push($colores, "rosa"); array_pop($colores); // elimina el último in_array("rojo", $colores); // true/false array_keys($persona); // ["nombre","edad","ciudad"] array_values($persona); // ["Luis", 28, "Medellín"] array_merge($colores, ["negro"]); // ── FOREACH CON CLAVE Y VALOR ───────────── foreach ($persona as $clave => $valor) { echo "$clave: $valor\n"; } ?>

Superglobales

Las superglobales son variables predefinidas por PHP accesibles desde cualquier ámbito (funciones, clases, archivos). Contienen información del servidor, la solicitud HTTP, las sesiones, las cookies y más. Son arrays asociativos que PHP llena automáticamente.
VariableDescripción
$_GETParámetros enviados por URL (método GET)
$_POSTDatos enviados por formulario (método POST)
$_REQUESTCombina $_GET, $_POST y $_COOKIE
$_SERVERInfo del servidor y del entorno de ejecución
$_SESSIONVariables de sesión del usuario actual
$_COOKIECookies enviadas por el navegador
$_FILESArchivos subidos mediante formulario
$_ENVVariables de entorno del sistema
$GLOBALSTodas las variables globales del script
superglobales.php
<?php // ── $_GET ──────────────────────────────── // URL: pagina.php?nombre=Ana&edad=30 $nombre = $_GET["nombre"] ?? "Invitado"; echo "Hola, " . htmlspecialchars($nombre); // ── $_POST ─────────────────────────────── // Formulario con method="post" if ($_SERVER["REQUEST_METHOD"] === "POST") { $email = $_POST["email"] ?? ""; echo "Email recibido: " . htmlspecialchars($email); } // ── $_SERVER ───────────────────────────── echo $_SERVER["SERVER_NAME"]; // ej: localhost echo $_SERVER["REQUEST_URI"]; // /ruta/pagina.php echo $_SERVER["REMOTE_ADDR"]; // IP del cliente // ── $_SESSION ──────────────────────────── session_start(); // ¡siempre al inicio! $_SESSION["usuario"] = "Carlos"; echo $_SESSION["usuario"]; // persiste entre páginas // ── $_COOKIE ───────────────────────────── setcookie("tema", "oscuro", time() + 86400); echo $_COOKIE["tema"] ?? "claro"; // ── $_FILES ────────────────────────────── // Formulario: enctype="multipart/form-data" $archivo = $_FILES["foto"]["name"]; $tmp = $_FILES["foto"]["tmp_name"]; move_uploaded_file($tmp, "uploads/" . $archivo); ?>
⚠️ Seguridad: Nunca uses datos de superglobales directamente en HTML o consultas SQL sin sanitizarlos. Usa siempre htmlspecialchars() para salida HTML y sentencias preparadas para bases de datos.
